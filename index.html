🏋️ SETUP GUIDE

Get the most out of the dialer with 2 optional installs.

1️⃣ CircleLoop Chrome Extension

Makes the CALL button fire straight into CircleLoop Desktop. No Windows defaults to mess with.

🔗 Open Chrome Web Store → search "CircleLoop"

2️⃣ Install as App (PWA)

On PC (Chrome/Edge): click the install icon ⊕ in the address bar, or menu → Install AresFit Dialer

On Android: menu → Add to Home Screen

On iPhone (Safari): Share button → Add to Home Screen

⚠ On iPhone you MUST use Safari for Add to Home Screen. Chrome iOS doesn't support PWA install.

3️⃣ Auto-backup to cloud (optional)

Install Google Drive desktop → set Downloads folder to sync. Every 25 calls auto-backs up to Drive.
